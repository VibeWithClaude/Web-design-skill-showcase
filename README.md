# Webdesign Skill Showcase: Starter Kit

This repository is a **high-performance boilerplate** designed to serve as a canvas for AI Agents to demonstrate their "Web Design Skills" and frontend engineering capabilities. 

Instead of being a finished gallery, this is a **"Ready-to-Build" environment** optimized for modern web standards including Next.js 16 and Tailwind CSS 4.

## 🎯 Purpose

This project provides the technical foundation for an AI Agent to:
1. **Demonstrate Design Skills:** Transform a blank canvas into a premium digital interface.
2. **Apply Design Systems:** Implement strict rules for typography, spacing, and materiality.
3. **Showcase Engineering:** Build performant, hardware-accelerated components and motion physics.

## 🚀 Environment Specs

- **Core:** Next.js 16 (App Router, Turbopack)
- **Framework:** React 19
- **Styling:** Tailwind CSS 4 (Beta/Native Support)
- **Agent Integration:** Pre-configured with the `webdesign-frontend-skill` skill.

## 🛠️ Usage for Users

1. **Install:** Run `npm install`.
2. **Launch:** Run `npm run dev`.
3. **Build:** After installation, you can start prompting the agent to build a website or show you its design skills.

## 🧭 Choose Your Path

Depending on your goal, you can engage with the AI Assistant in two different ways:

### 🌈 Path A: The Experimenter (Discovery Mode)
*Best for: Quickly seeing the AI's design taste and technical capabilities.*

**Prompt the Agent:** Ask your AI Assistant to "Show me your design taste" or "Build a premium landing page using your frontend skill."

OR 

Simply use the **Quick Start Prompt** in the section below. This will trigger the AI to build a high-end, interactive showcase using its predefined design system. 


### 🏗️ Path B: The Business Builder (Production Mode)
*Best for: Creating a professional, custom website for a business or brand.*

Instead of one-off prompts, initiate a **Discovery Interview**. Ask the Assistant to:

```
I want to build a professional website for my business. Can you start a Discovery Interview with me to define the requirements?

**During this process, ask me to provide you with the following information:**
- **Inspiration:** Links or descriptions of websites you admire.
- **Brand Assets:** Logos, color palettes, and specific high-res images.
- **Context:** Business descriptions, target audience, and core value propositions.
- **Structure:** Desired layouts, number of pages, and specific features (e.g., contact forms, galleries).
- **Documents:** Content, copy, or technical specs for the business.

Record Interview in the `interview.md` file and then use the `` skill to build the website.
```

## ✨ Example Agent Session

This environment is designed to turn a simple prompt into a premium digital experience. Below is an example of what you can achieve by collaborating with an AI Agent in this workspace.

### 🚀 Quick Start Prompt
Copy and paste the following prompt to have the AI Agent build the "Aether Design Lab" showcase:

> "Build a premium, high-end agency landing page called 'Aether Design Lab' using your 'webdesign-frontend-skill' skill. Incorporate an asymmetric Hero section with Geist typography, a Bento 2.0 grid for features, and an 'Interactions Lab' section featuring magnetic spring-based physics, 3D tilt cards, and a morphing Dynamic Island component. Use a Zinc-950 and Deep Rose color palette and ensure all animations use high-performance hardware-accelerated transforms."

### 📈 What to Expect
- **Visual Polish:** A clean, minimal dark aesthetic (Zinc-950) avoiding generic AI "glow" clichés.
- **Mechanical Physics:** Buttons that magnetically pull toward the cursor and cards that tilt in 3D space.
- **Modern Stack:** Full utilization of Next.js 16, Tailwind 4, and Framer Motion for liquid-smooth transitions.

## 🤖 For Agents

This workspace expects the Agent to adhere to the `webdesign-frontend-skill` instructions located in `.agent/skills/design-skill/SKILL.md`. This includes using premium typography stacks, avoiding AI-cliché purples/neons, and implementing spring-based motion physics.
